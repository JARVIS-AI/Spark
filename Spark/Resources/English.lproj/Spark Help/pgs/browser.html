Items Browser

There are 3 kinds of items. HotKeys that can be create and edited in the main windows, Actions & Applications. To create and edit Actions & Applications you need open the Items Browser.

You can create lists to sort your Actions & Applications, and you can create, edit and delete Actions & Applications.

Note: HotKey Lists and Actions Lists don't have any functions others than sorting items, but Applications Lists are useful to group Applications and associate an Action to a whole group instead of each Application.

To create a list, you can click on the button. Fill the name field in the window and click "Add" button.

Double click on a list to edit its name.

To add Items in a list, drag and drop them on the list row.

To remove Items from a list, select them, and type delete or suppr on your keyboard.

To add new Applications you can either add them with the open panel provided by Spark, or drop Applications icons directly in the Table.
