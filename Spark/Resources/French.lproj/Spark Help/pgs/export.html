Exporter & Importer des Items

Spark ne permet pas d'exporter des objects à l'unité, mais il permet cependant d'exporter une liste d'objects. Quand vous importez une liste d'objets, Spark permet de choisir quels objects vous désirez importer, et quels objects vous désirez laisser.

Il est également possible d'exporter une liste de Raccourci au format HTML, ce qui peut s'avérer pratique si vous voulez imprimer le liste des raccourcis que vous avez définis.

Pour exporter une liste simple, selectionnez le liste à exporter et choisissez "Exporter la liste sélectionnée…" dans le menu Fichier.

Pour importer une liste, ouvrez là avec Spark.
