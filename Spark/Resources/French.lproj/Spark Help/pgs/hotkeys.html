Raccourcis & Actions

Spark utilise principalement deux types d'objects: Les Raccourcis, et les Actions.

Raccourci:

Un raccourci représente une combinaison de touche. Vous pouvez créer deux type de raccourcis. Les raccourcis Simple qui execute une action quand vous tapez le raccourcis, et les raccourcis Avancés qui execute différentes actions en fonction de l'application au premier plan.

Actions:

Une Action définie l'action à executer lorsqu'un raccourci est executé. Un raccourci doit obligatoirement contenir au moins une Action, l'Action par default.
Pour en savoir plus sur les Actions, lisez "Les Différents types d'Actions"

Application:

Spark défini aussi des Applications. Quand vous créez un Raccourci Avancé, vous pouvez associé une action à une Application ou à un groupe d'applications. Ces objects Applications peuvent-être regroupé en listes, et si vous associez une Action à une liste, elle sera executée si n'importe laquelle des applications contenu dans la liste est au premier plan, lorsque que vous tapez le raccourci.
Spark définit une Application spéciale: "Système (defaut)". Cette application représente l'action par défaut executée par un raccourci.
