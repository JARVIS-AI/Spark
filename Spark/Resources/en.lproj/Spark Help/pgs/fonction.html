How Does Spark Work ?

Spark contains two applications. The First one, "Spark", is a Hot Keys editor. It allows creating and managed Hot Keys. You can launch it by double clicking on Spark Icon. However, if Spark weren't "Active" this Hot Key wouldn't work. To activate Spark, you need launch a second Application called "Spark Daemon".

"Spark Daemon" is an invisible background application. It waits for Hot Keys events and when it catches one, executes the action you set with "Spark". You can set in Spark's preferences that "Spark Daemon" start at launch, i.e. Spark will be activated when you login and your Hot Keys will be enabled. You can also launch and stop "Spark Daemon" in Spark.

To activate Spark, you can choose menu File > Activate Spark, or click the button located in the top of Spark's main window.

You can check Spark state at bottom of Spark's main window or by looking on the start/stop button icon.
