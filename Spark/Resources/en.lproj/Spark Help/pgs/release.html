Release Notes

Version 2.1

Spark 2.1
• Improve Interface and VoiceOver Support.
• Enable One Key Shortcut support (show Spark Preferences).
• Minor bugs corrections.
Plugins
• Add a new Keystroke plugin that allows recording a serie of Keystroke to execute.
• Add "Launch/Quit" application action that switch application state.
• Add "Hide others" application action.
• Add "Screen Saver" Power action (Tiger only).
• Update "Switch Visual" in iTunes plugin (one iTunes update broke it).
