Backup & Restore a Library

If you work one different machines, it is useful to have the possibility to keep a Spark Library, and don't have to recreate it on each machine. You can do this by exporting your Library. Note that some actions can be broken after an importation. For example, an Action configure to open an application, will not works if the application is not installed on the target computer.

When you try to import a Library, Spark allows restoring the Library or merges it. If you choose Restore, Spark will destroy all items currently in your Library and replace them by the full contents of the imported Library.

If you choose to merge, Spark let you choose in a list the items you want to import.

To export your Library, choose File > Backup Library….

To import a Spark Library File, double clic on it or drag it on the Spark Icon.
