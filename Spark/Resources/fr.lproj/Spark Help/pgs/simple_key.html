Créer un Raccourcis Simple

Si vous n'avez pas besoin des possibilités avancées de Spark, il est facile de les ignorer. Le panneau d'édition des Raccourcis contient un boutton () dans la barre d'outil qui permet de passer du mode "Simple" au mode "Avancé".

En Mode Simple le panneau affiche un éditeur d'action. Vous pouvez choisir le type d'action que vous désirez à l'aide du menu "Catégorie".
Ensuite vous devez choisir une combinaison de touches configuer l'action choisie.
L'éditeur d'Action dépend du plug-in que vous utilisez. Pour plpus d'aide sur un plugin, choisissez "Ouvrir l'aide des Modules" dans le menu aide de Spark.

Quand vous créer un Raccourcis, vous devez choisir une combinaison de touches. Pour choisir une combinaison, sélectionez le Champs "Raccouci" et tapez mes touches que vous souhaitez enregister.

Il n'y a pas de restriction partculière dans le choix d'un raccourci, il faut donc faire attention de ne pas sélectionner un raccourci déjà utilisé par une autre application, ou le raccourci de Spark primera sur ce dernier. Normalement un raccourci nécessite au moins deux touches excepté pour les touche F1 à F16, qui peuvent être utilisées seul.

Pour créer un raccourci, vous pouvez:

  • Cliquer sur le bouton et choisir le type de raccourci que vous désirez.
  • choisir un raccourci dans le menu Fichier > Nouveau raccourci
  • double cliquer sur la liste correspondante au type de raccourci que vous voulez créer

Pour éditer un raccourci, double cliquez dessus.

Pour supprimer un ou plusieurs raccourcis d'une liste, sélectionnez les et appuyez sur la touche Effacer ou la touche Supprimer.

Pour supprimer définitivement un raccourcis, effacez le de la bibliothèque ou de la liste du module correspondant au raccourci.
