Raccourci AppleScript
Spark
Qu'est-ce qu'un Raccourci AppleScript?

Un "Raccourci AppleScript" est un raccourci Spark qui permet, comme son nom l'indique, d'executer un Apple Script.
Vous avez la possibilité d'enregister un script dans un raccourci pour l'executer chaque fois que vous en aurez besoin, ou vous pouvez également associer le raccourci à un fichier de script compilé (fichier .scpt) qui s'executera sur demande.
Le module AppleScript inclus un éditeur de script simplifié.

Comment créer un raccourci AppleScript?

Il y a deux possibilités pour créer un raccourci AppleScript. La première, vous pouvez utiliser l'éditeur pour saisir le script que vous voulez executer.

Le boutton "Compiler" permet de vérifier la syntaxe du script.
Le bouton "Executer" permet d'essayer votre script. Il vérifiera la syntaxe avant l'execution du script.
Le boutton "Ouvrir" permet d'importer le contenu d'un fichier texte ou d'un script dans l'éditeur.
Le boutton "Éditeur" permet de lancer l'"Éditeur de Script" Apple et d'ouvrir le script que vous avez saisi dans le champs de texte.

Si vous désirez executer un grand script ou un script déjà compilé, il faudra utiliser la seconde technique.

Cliquez sur l'onglet "Fichier" et ensuite choisissez le script que vous voulez executer. Le fichier de script sélectionné doit obligatoirement être un script déjà compilé. Si vous désirez executer un gros script, il est préférable de le compiler au préalable et de l'enregister dans un fichier. Vous pouvez utiliser l'"Editeur de Script" Apple pour réaliser cette tâche.

Le type de raccourci qui sera crée dépendra de l'onglet sélectionné au moment ou vous déciderez d'enregister le raccourci.
