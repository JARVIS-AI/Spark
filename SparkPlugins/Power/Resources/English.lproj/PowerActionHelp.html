What is a Power Action?

A "Power Action" is a simple key with Action:

  • Log Out…
  • Fast Log Out… (switch to login window)
  • Sleep
  • Reboot
  • Shutdown

How to create a Power Action?

Creating a Power Action is pretty simple. You just select the Action you want to execute on the Action Menu and that's all.

Be carful with those keys. Only the logout key will ask you to confirm the Action when it is executed.
