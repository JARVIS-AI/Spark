What is an iTunes Action?

A "System Action" allows you to execute the following actions:

  • Launch iTunes (and start to play).
  • Quit iTunes.
  • --------
  • Play/Pause.
  • Previous Track.
  • Next Track.
  • Stop.
  • Show Track Info.
  • --------
  • Play Playlist…
  • Rate current track…
  • --------
  • Volume Down/Up.
  • --------
  • Start/Stop iTunes Visual.
  • Eject Audio CD.
