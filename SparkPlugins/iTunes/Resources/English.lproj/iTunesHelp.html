What is an iTunes Action?

An "iTunes Action" is a Spark Action made to command iTunes. You can create Actions that Launch or Quit iTunes, Actions that play/pause the current track, Actions to go to the Previous or the Next track, Volume up or down, and Actions that start/stop iTunes visual.

How to create an iTunes Action?

Creating an iTunes Action is really simple, but those keys are my favorites. You just select the Action you want to execute on the Action Menu and that's all.
