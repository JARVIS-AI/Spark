How to create an iTunes Action?

Creating an iTunes Action is pretty simple. Select the Action you want to execute on the Action Menu and that's all.

Actions Options:
Launch iTunes…
- Launch and hide: Hides iTunes.
- Launch in background: iTunes stays in background.
- Display iTunes Icon: Display a feedback.
- Start playing: iTunes starts to play.
Play/Pause, Previous/Next Track
- Show Track Info: Display track info (visuals settings).
