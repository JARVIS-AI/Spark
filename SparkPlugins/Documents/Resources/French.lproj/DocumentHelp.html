Qu'est ce qu'un raccourci de Document?

Un "Raccourci de Document" est un raccourci qui permet d'ouvrir un dossier ou un document avec l'application par défaut ou avec un application choisie. Vous pouvez également créer un raccourci pour ouvrir les éléments sélectionnés dans le Finder.

Comment créer un raccourci de Document?

Si vous voulez ouvrir un document ou un dossier avec l'application par defaut pour ce type de fichier, il suffit de sélectionner "Ouvrir" dans le meun Action et de cliquer sur le bouton "Choisir…" pour sélectionner le fichier/dossier à ouvrir.
Si vous voulez que le document sélectionné s'ouvre dans un application spécifique, choisissez "Ouvrir avec" dans le menu "Action". Le menu qui apparait contient les applications susceptibles d'ouvrir le document sélectionné. Si l'application que vous recherchez ne se trouve pas dans le menu, selectionnez "Choisir…" dans le menu pour l'ajouter.
