What is a Document Action?

A "Document Action" is a Spark Action that allows opening a Folder or a Document with default Application or with a specified Application. You can also create an Action that opens selected files in the Finder with a specified application.

How to create a Document Action?

If you want open a Folder or a document with its default application, just select "Open" in the Action Menu.
If you want to force an application for open a file, you can select "Open With…". When you choose a document, the menu "With Application" is filled with the applications that can open this document. If the Application you want is not in the menu, you can add it by selecting the "Choose…" item in this menu.
