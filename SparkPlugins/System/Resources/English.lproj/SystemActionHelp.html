What is a System Action?

A "System Action" allows you to execute the following actions:

  • Log Out…
  • Fast Log Out… (switch to login window)
  • --------
  • Sleep
  • Reboot
  • Shutdown
  • --------
  • Switch grayscale.
  • Switch black/white.
  • --------
  • Launch Screen Saver (Mac OS X.4 only)

How to create a System Action?

To create a «System Action» you just have to select the action you want to execute in the Action Menu.
