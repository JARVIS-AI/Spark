Raccourci d'Application
Spark
Qu'est ce qu'un raccourci d'Application?

Un "Raccourci d'Application" est un raccourci créer spécialement pour lancer une application.
Vous pouvez également utiliser un raccourci de Document pour lancer une application mais le raccourci d'Application a plusieurs avantages.

• Il permet de modifier le comportement de l'application lors de son lancement.

• Il utilise un mécanisme légerement différent pour stocker le chemin de l'Application. Si vous réalisez une manipulation qui d'ordinaire rend un alias invalide (déplacer l'application sur un autre volume par exemple), le raccourci d'Application essayera de la retrouver alors qu'un raccourci de Document ne pourra pas.

Comment créer un raccourci d'Application?

Pour créer un raccourci d'Application, il suffit de choisir "Application" dans le menu "Nouveau raccourci" ou de double cliquer sur la liste Application.
Ensuite vous pouvez choisir un application en cliquant sur le bouton "Choisir…".
Vous avez le choix entre 3 actions:

  • Démarrer: Démarre un application ou la ramène au premier plan si elle est déjà lancée. Cette action à le même effet que double-cliquer sur l'icone de l'application.

  • Quitter: Demande à l'application cible de quitter. Cette action à le même effet que de sélectionner "Quitter" dans le menu de l'application.

  • Forcer à quitter: Utilisez cette action avec précaution. Elle à le même effet que de sélectionner "Force à quitter" dans le menu Pomme.

Si vous choisissez l'option démarrer vous avez 4 options:

 - "Laisser en arrière plan": L'application reste en arrière plan après son lancement.
 - "Masquer les autres": Un fois l'application démarrée, les autres applications sont masquées.
 - "Masquer après démarrage": Masque l'application quand elle a fini de démarrer.
 - "Démarrer une nouvelle instance": Si l'application n'est pas démarrée, lance l'application normalement. Si elle est déjà lancée, au lieu de la ramener au premier plan, le raccourci lance une nouvelle copie de l'application. Cette option ne fonctionne pas avec toutes les applicatons.
