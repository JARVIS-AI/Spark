Les différents types d'Actions

Spark fonctionne à l'aide de modules. Chaque module définit un type d'Action. Les modules Spark sont reconnaissables grâce à leur extension «.spact».

Spark contient un certain nombre de modules, mais il est également possible d'en ajouter en les plaçant dans votre Bibliothèque.

Pour installer un module dans Spark, double cliquer dessus dans le Finder, et Spark vous porposera de l'installer

Pour ajouter un module à Spark, placez le dans le dossier "Application Support : Spark : PlugIns" qui se trouve soit dans votre bibliothèque utilisateur pour qu'il ne soit accessible que par vous, soit dans la Bibliothèque principale pour que tout les utilisateurs puissent en profiter.

Parmis les modules livrés avec Spark, on trouve:

  • Les Actions d'Application
  • Les Actions de Documents
  • Les Actions d'Énergie
  • Les Actions iTunes
  • Les Actions AppleScript

Pour en savoir plus sur ces Actions, consulter le menu Aide > Aide des Modules dans Spark.
