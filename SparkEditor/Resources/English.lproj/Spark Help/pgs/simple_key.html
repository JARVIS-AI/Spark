Creating a Simple HotKey

If you don't need Advanced Capability of Spark, you can easily ignore them. The HotKey editor panel contains a toolbar item () that allows switching between Simple Mode and Custom Mode.

Simple Mode displays a panel with an Action Editor. You can choose the kind of Action you want with the "Categorie" popup menu.
Then you need just choose a keystroke, and configure your action.
The Action configuration panel depends of the kinds of Action you want to create. To get help, open Plugins Help in Spark Help Menu.

When you create a Hot Key, you must choose a key combination. To do this, select the shortcut field and type the combination you want to record.

There is no particular limitation in the combination you choose, but be careful, try to don't use a system shortcut. Normally, all combinations must contain at least two keys, except for Function Keys (F1 to F16) that can be use alone.

To create a new Hot Key you can:

  • Click on the button and choose what kind of Hot Key you want.
  • Choose a kind of Hot Key in the File > New Hot Key menu
  • Double click on a plug-in list in the Spark window.

To edit a Hot Key, double click on it, or select it and press Return.

To remove one or many Hot Keys from a list, select them and type delete or suppr on your keyboard.

To delete one or more Hot Keys, select them in the Library list or in a plug-in List, and type delete or suppr.
