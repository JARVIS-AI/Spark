Export & import Items List

Spark doesn't provide a way to export single object. But you can export a List of Items. When you import this list, Spark let you choose the items you want to add to the Library.

You can also export a HotKey List in HTML format. It is very useful if you want to print a list of shortcuts you defined with Spark.

To export a single List, select the targeted list a choose File > Export Selected List….

To import a Spark List File, double click it or drag it on the Spark Icon.
